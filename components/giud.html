什么是前端？前端能实现什么？ 前端即网站前台部分，运行在PC端，移动端等浏览器上展现给用户浏览的网页。随着互联网技术的发展，HTML5，CSS3，前端框架的应用，跨平台响应式网页设计能够适应各种屏幕分辨率，合适的动效设计，给用户带来极高的用户体验。

简单的来说，我们日常在电脑，手机上浏览的页面，你随手逛的淘宝，看的每一个图片，每一段文字，每一个优美的画面都是前端来实现的…例如下图，是不是很常见…,当然前端能实现的不仅仅是这些简单可视的东西，接下来我们一起学习吧(╹▽╹)

什么是HTML HTML 是用来描述网页的一种语言。HTML 是一种在 Web 上使用的通用标记语言。HTML 允许你格式化文本，添加图片，创建链接、输入表单、框架和表格等等，并可将之存为文本文件，浏览器即可读取和显示。

HTML 指的是超文本标记语言: HyperText Markup Language HTML 不是一种编程语言，而是一种标记语言 标记语言是一套标记标签 (markup tag) HTML 使用标记标签来描述网页 HTML 文档包含了HTML 标签及文本内容 HTML文档也叫做 web 页面

通俗的来讲，我们看到的每一段文字，每一个图片，都是通过HTML的不同类型的标签来实现的，文字的大小，颜色，图片的样式，位置等等，是通过css来实现的。
