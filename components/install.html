# 下载vscode

下载地址：Download Visual Studio Code (opens new window)

TIP

选择对应的版本下载

Windows系统和Mac系统

# 安装vscode

第一步：勾选同意协议

第二步：选择安装位置（不推荐安装在C盘）

第三步：点击下一步

第四步：勾选红框内已购选到的

第五步：点击安装

第六步：等待安装

第七步：安装完成打开vscode 配置中文,搜索Chinese 点击install

第八步：点击reset 然后重启vscode就OK了


# 下载谷歌浏览器

第一步：在电脑浏览器中打开下载地址谷歌浏览器 (opens new window) 打开谷歌官方网站

第二步：进入官网后单击“下载Chrome”

第三步：3、双击打开文件，等待安装即可（如果中间弹出什么窗口，就点击“运行”或“是”或"允许安装"，反正一路同意）

第四步、安装完成后，单击Chrome浏览器右上角的“…”，然后单击设置，打开设置页面

第五步、进入设置页面后选择外观，按照下图配置：显示主页、主页打开百度：https://www.baidu.com/、显示书签栏

第六步：在搜索引擎中设置搜索引擎为百度

第七步：常见设置到此结束，其他功能自行探索
